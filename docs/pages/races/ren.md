# 人族

## 种族定位

人族是最适合新手的种族，拥有减伤被动能力，在险峻环境中生存能力较强。人族主宰大陆大部分地区，与仙族关系友好，与妖族长期冲突。

## 种族祝福

**女娲的守护**（抗性提升Ⅰ）

- 效果：减免部分伤害
- 定位：提高生存容错率，适合新手玩家

## 背景故事

女娲用泥土和水创造了人类，赐予其生命与创造力。人族凭借快速繁衍与领土扩张，持续发展文明。因自然环境开发，与崇尚自然的妖族冲突不断，局势日益紧张。人族与仙族关系友好，两族共同建造了皇城。

## 引路人对白

## 新手流程

1. 出生点找到引路人小仁（1677 134 138），用种族证明兑换推荐书
2. 沿阶梯往上走，隧道前右转沿路往上，找到族长家（1690 143 113）
3. 族长处兑换新手装备兑换券
4. 前往铁匠铺（1739 160 103）兑换职业武器、防身短匕、初心者套装
5. 前往丹药铺（1655 153 166）兑换新手疗愈丹
6. 通过传送点（1600 130 123）前往皇城
