# 妖族

## 种族定位

妖族拥有全种族最快的移动速度，适合需要高机动性的玩法。崇尚自然，拥有最齐全的历史记录。

## 种族祝福

**世界的意志**（迅捷效果）

- 效果：跑得比其他种族快
- 定位：高机动性，快速穿梭各城镇，走位灵活

## 背景故事

妖族集日月精华所生，崇尚自然，认为万物应遵循世界的规则与循环。近年来妖族居住范围饱受人族侵略，仙族建造镇妖塔关押上千名妖族同伴。妖族虽能力强大，但敌不过人仙两族联手，活动空间越来越小。族内大长老失踪并发来求救信息，局势沉重。

## 新手流程

1. 出生点找到引路人小花（2848 48 888），用种族证明兑换推荐书
2. 前往古树前水池边找到叶灵谷谷主（2788 53 863）
3. 谷主处兑换新手装备兑换券
4. 前往铁匠铺（2653 96 858）兑换职业武器、防身短匕、初心者套装
5. 前往丹药铺（2663 89 895）兑换新手疗愈丹
6. 通过传送点（2850 53 830）前往皇城

**注意**：妖族传送点位于皇城外面，需前往东城门（340 45 65）进入皇城。

## 种族关系

- 友好：战神族
- 敌对：人族（领土被侵略）、仙族（镇妖塔）
- 特殊：拥有五族中最齐全的历史记录
