# 地图与区域

## 设计概述

游戏世界由五大种族领地与中央皇城组成。各区域通过传送点网络连接，玩家可在种族领地与皇城之间快速移动。每个区域设有功能性设施满足不同需求。

## 区域总览

| 区域 | 地形特征 | 所属种族 | 核心功能 |
|------|----------|----------|----------|
| 人族领地 | 地面平原 | 人族 | 新手引导、基础设施 |
| 仙族领地 | 高山 | 仙族 | 新手引导、基础设施 |
| 神族领地（昆仑） | 空岛 | 神族 | 新手引导、基础设施 |
| 妖族领地（叶灵谷） | 地面森林 | 妖族 | 新手引导、基础设施 |
| 战神族领地 | 地下熔岩 | 战神族 | 新手引导、副本入口 |
| 皇城 | 城市 | 公共 | 主线推进、商业、社交 |
