# 五行总览

## 设计概述

五行系统是盘灵古域的核心属性体系。金、木、水、火、土五种元素贯穿战斗、炼丹、世界观等多个系统。五行之间存在相生相克关系，影响伤害计算与丹药配方。

## 五行属性表

| 元素 | 属性 | 对应圣兽 | 方位 | 特质 |
|------|------|----------|------|------|
| 金 | 刚毅锋锐 | 白虎 | 西 | 攻击、破防 |
| 木 | 生长自然 | 青龙 | 东 | 生命、恢复 |
| 水 | 流动智慧 | 玄武 | 北 | 防御、减速 |
| 火 | 炽热力量 | 朱雀 | 南 | 爆发、灼烧 |
| 土 | 厚重承载 | — | 中 | 稳固、控制 |

## 相生关系

五行相生循环：

木 → 火 → 土 → 金 → 水 → 木

- 木生火：木燃烧生火
- 火生土：火焚烧成灰为土
- 土生金：土中蕴藏金属
- 金生水：金属凝结水珠
- 水生木：水滋养草木

## 相克关系

五行相克循环：

金 → 木 → 土 → 水 → 火 → 金

- 金克木：金属砍伐草木
- 木克土：草木破土而出
- 土克水：土能阻挡水流
- 水克火：水能灭火
- 火克金：火能熔化金属

## 元素在炼丹中的应用

炼丹配方中元素必须按**金→木→水→火→土**的固定顺序放入炼丹炉。不同丹药需要不同的元素组合，但顺序不可改变。

## 元素获取

- 野外采集
- 击杀敌人掉落（炼丹师额外掉落）
- 商店购买

## 实现要点

- 五行相生相克伤害加成/减免公式
- 元素属性附加到武器/技能
- 炼丹配方的元素顺序验证
- 四圣兽与五行的关联系统
