# 任务系统

## 设计概述

任务系统以主线剧情引导玩家从种族出生地逐步前往皇城，途中学习游戏基础系统。任务通过NPC对话泡泡道具触发交互，完成后获得奖励与剧情推进。

## 任务类型

| 类型 | 说明 | 触发方式 |
|------|------|----------|
| 新手引导 | 教学基础操作与系统 | 自动/引路人 |
| 主线剧情 | 推进核心故事线 | 剧情书/NPC对话 |
| 知识试炼 | 考验世界观知识 | 进入副本区域 |
| 阵营战役 | 大规模PvP活动 | 特定NPC发起 |

## 主线任务流程

### 第一阶段：种族出生地

1. 找到引路人，兑换推荐书
2. 前往族长处报到
3. 兑换新手装备兑换券
4. 铁匠铺兑换职业武器与初心者套装
5. 丹药铺兑换新手疗愈丹

### 第二阶段：前往皇城

6. 通过种族传送点前往皇城
7. 皇城内接续主线剧情

### 第三阶段：皇城主线

8. 探索皇城各功能区域
9. 接取后续任务与挑战

## 任务交互机制

### NPC对话泡泡

- 用途：向NPC接取任务的消耗道具
- 消耗：每次交易消耗一枚对应道具
- 补充：不足时可在菜单书中重新领取

### 剧情书系统

- 主线剧情以书本形式呈现
- 阅读完毕后解锁下一步任务
- 各种族有独立的剧情书内容

## 实现要点

- 任务状态机：接取→进行中→完成
- 对话泡泡消耗与补充逻辑
- 剧情书阅读追踪
- 青龙试炼答题判定与全队重置
- 逆天计划战役的发起与参与条件
