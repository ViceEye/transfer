# 弓箭手

## 职业定位

弓箭手是在队伍后方进行远程攻击的职业。虽然护甲血量不高，但攻击力十分强大。需要队友掩护或依靠走位技巧来保证生存。

## 分支系统

### 游侠

| 属性 | 说明 |
|------|------|
| 定位 | 灵活型远程 |
| 攻击力 | 较低（相对狙击） |
| 移动速度 | 快 |
| 攻速 | 较快 |
| 特殊能力 | 同时射出三只箭矢 |
| 适合场景 | 在敌阵中穿梭、需要高机动性的战斗 |

游侠武器攻击力相对较低，但移动速度快、攻速快，且能同时射出三只箭矢。适合在敌阵中穿梭，需要更多技巧来回避敌人攻击。

### 狙击

| 属性 | 说明 |
|------|------|
| 定位 | 爆发型远程 |
| 单次伤害 | 全体职业最高 |
| 移动速度 | 较慢 |
| 生命值 | 较低 |
| 适合场景 | 有队友掩护时的后排输出 |

狙击武器单次伤害是全体职业中最高的，但速度较慢、生命值较低。在有队友掩护的场合，狙击将是相当强悍的角色。

## 分支对比

| 对比项 | 游侠 | 狙击 |
|--------|------|------|
| 单次伤害 | 低 | 全职业最高 |
| 攻速 | 快 | 慢 |
| 移动速度 | 快 | 慢 |
| 生命值 | 中 | 低 |
| 多目标能力 | 强（三箭） | 弱 |
| 依赖队友 | 低 | 高 |
| 操作难度 | 高（走位） | 中（站桩） |

## 实现要点

- 远程射击弹道计算
- 三箭矢同时发射的扇形判定
- 攻速与移动速度的分支差异
- 后排站位与仇恨管理
