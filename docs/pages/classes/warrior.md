# 战士

## 职业定位

战士是站在队伍最前方的职业，负责吸引敌人注意力并保护队友。拥有较高的生存能力与近战伤害，是团队的核心前排。

## 分支系统

### 金钟

| 属性 | 说明 |
|------|------|
| 定位 | 防御型前排 |
| 生存力 | 较高增幅 |
| 特殊能力 | 横扫攻击（对周围怪物造成伤害） |
| 适合场景 | 面对成群敌人、队友掩护下的前排站位 |

金钟武器赋予使用者较高的生存力增幅，并能使出横扫攻击对目标周围的怪物同样造成一定伤害。适合需要面对大量敌人的场合。

### 破军

| 属性 | 说明 |
|------|------|
| 定位 | 进攻型前排 |
| 移动速度 | 更快 |
| 攻击力 | 更高 |
| 特殊能力 | 无横扫攻击 |
| 适合场景 | 击破落单敌人、冲出重围 |

破军武器移动速度更快、攻击力更高，但没有横扫攻击。更适合击破落单的敌人或冲出重围，注重使用者的走位与技巧。

## 分支对比

| 对比项 | 金钟 | 破军 |
|--------|------|------|
| 生存力 | 高 | 中 |
| 攻击力 | 中 | 高 |
| 移动速度 | 普通 | 快 |
| 群战能力 | 强（横扫） | 弱 |
| 单挑能力 | 中 | 强 |
| 操作难度 | 低 | 高 |

## 实现要点

- 横扫攻击范围判定与伤害计算
- 仇恨吸引机制
- 金钟/破军武器属性差异化
- 移动速度加成系统
