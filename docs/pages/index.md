# 策划案总纲

## 项目简介

盘灵古域是一款以中国古代神话为背景的多人角色扮演游戏。游戏构建了一个由五大种族共存的盘古大陆，融合五行元素体系、多职业战斗系统与丰富的任务剧情。玩家选择种族与职业后，在开放世界中探索、战斗、炼丹、完成任务，体验东方奇幻冒险。

## 世界观概述

盘古开天辟地后，其后裔神族统领四圣兽（青龙、白虎、朱雀、玄武）守护世界秩序。女娲造人，人族繁衍扩张主宰大陆。部分人类修炼飞升成为仙族，建城于高山之上。集日月精华而生的妖族崇尚自然，与人族冲突不断。隐秘的战神族来源不明，拥有强大战斗力。五族之间的恩怨纠葛构成了游戏的核心叙事。

## 种族系统概述

游戏设有五大可选种族，每个种族拥有独特的种族祝福（被动能力）、专属出生地与背景故事。种族选择影响玩家的初始属性加成和游戏体验风格，但不限制职业选择。需实现种族选择流程、种族祝福效果、各种族出生点与新手引导。

## 职业系统概述

三大基础职业：战士（近战坦克）、弓箭手（远程输出）、炼丹师（辅助/控制）。每个职业拥有两个分支路线，提供不同的战斗风格。炼丹师额外拥有完整的炼丹制作系统。需实现职业选择、分支武器差异化、炼丹系统完整流程。

## 五行系统概述

金、木、水、火、土五种元素构成游戏的核心属性体系。五行相生相克关系影响战斗伤害计算与炼丹配方。四圣兽分别对应四种元素与四个方位。需实现五行属性系统、相生相克伤害公式、元素采集与消耗机制。

## 战斗系统概述

基于职业分支的实时动作战斗，强调走位与团队配合。战士负责前排吸引仇恨，弓箭手后排输出，炼丹师提供治疗与控制。包含野外战斗、副本挑战（蚩尤试炼）、阵营对抗（逆天计划）等多种战斗模式。需实现仇恨机制、横扫攻击、远程射击、丹药即时释放等。

## 任务系统概述

以主线剧情引导玩家从种族出生地前往皇城，途中学习基础系统。包含新手引导任务、主线剧情任务、青龙试炼（知识问答副本）等。任务通过NPC对话泡泡道具触发交互。需实现任务接取/完成流程、剧情书系统、试炼副本机制。

## 地图概述

游戏世界包含五大种族领地与中央皇城。各种族领地设有出生点、铁匠铺/仙器铺、丹药铺、传送点等功能区域。皇城作为公共区域包含钱庄、市集、风花楼、护国殿等数十个功能场所。需实现区域传送系统、重生点记忆、城门进出逻辑。

## NPC系统概述

NPC承担任务引导、物品兑换、剧情推进等核心功能。每个种族配备引路人、族长、铁匠、丹药师等角色。皇城设有侍卫长、商人、茶坊老板等功能NPC。交互通过对话泡泡道具消耗实现。需实现NPC对话系统、兑换逻辑、对话泡泡消耗与补充机制。

## 物品装备概述

装备按职业分类，各职业武器不通用。新手装备包含职业武器、防身短匕、初心者套装。消耗品包含各等级丹药（药丹/法丹/术丹）、元素材料、药引。功能道具包含种族证明、NPC对话泡泡、重生石、菜单书等。需实现装备锻造台系统、丹药炼制系统、道具使用逻辑。

|------|----------|------------|
| 种族系统 | 五大种族、种族祝福、出生地 | 种族选择、被动效果、新手引导流程 |
| 职业系统 | 三职业六分支、炼丹制作 | 武器差异化、炼丹炉交互、丹药效果 |
| 五行系统 | 五元素相生相克、圣兽对应 | 属性计算、伤害公式、元素采集 |
| 战斗系统 | 实时动作、团队配合、多模式 | 仇恨机制、技能系统、副本逻辑 |
| 任务系统 | 主线引导、试炼副本、剧情书 | 任务状态机、对话触发、奖励发放 |
| 地图系统 | 五领地+皇城、传送点网络 | 区域加载、传送逻辑、重生点 |
| NPC系统 | 引路人、族长、商人、功能NPC | 对话树、兑换系统、泡泡消耗 |
